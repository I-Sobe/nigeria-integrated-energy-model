# Nigeria Integrated Energy-Finance Model

A proof-of-concept framework for analyzing the interactions between Nigeria's gas supply system, electricity generation fleet, transmission network, electricity market, and energy project economics.

The model combines operational performance indicators, fuel supply constraints, market settlement metrics, capacity factor analysis, and financial evaluation techniques to explore how technical and commercial conditions influence generation outcomes and investment viability.

Current development focuses on gas-fired generation, market liquidity, transmission constraints, renewable integration, and project finance applications within the Nigerian power sector.
